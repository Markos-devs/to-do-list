# 📝 To-Do List App

A simple and responsive To-Do List web app built with HTML, CSS, and JavaScript.

## Features

- ✅ Add new tasks
- 🗑️ Delete tasks
- ✅ Mark tasks as completed
- 💾 Saves tasks in browser (LocalStorage)

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Use

1. Clone the repo:
   ```bash
   git clone https://github.com/Markos-devs/to-do-list.git
   ```
